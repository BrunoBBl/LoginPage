# LoginPage
Página de Login Responsiva
/////////////////////////////////

Projeto: Página de login responsiva.

Tecnologia usada: Framework Bootstrap para estilização e responsividade.

Objetivo: Criar uma interface simples e funcional para autenticação de usuários.

Funcionalidades principais:

Campos de entrada para usuário e senha.

Uso de grupos de entrada do Bootstrap para melhor organização visual.

Navegação fluida entre elementos interativos (suporte a teclado).
